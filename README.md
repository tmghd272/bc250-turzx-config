## BC-250 Turzx Config (Arch/CachyOS)

Overlay configuration + setup script for [turing-smart-screen-python](https://github.com/mathoudebine/turing-smart-screen-python), optimized for BC-250 systems.

Currently designed for Arch/CachyOS-based systems only.

## Installation Turzx Toolkit

Clone the repository to the following path:

```
git clone https://github.com/tmghd272/bc250-turzx-config.git ~/Apps/bc250-turzx-config
```

Start the toolkit:

```
cd ~/Apps/bc250-turzx-config
chmod +x bc250-turzx-toolkit.sh
./bc250-turzx-toolkit.sh
```
